# SearchWith

選択した文字列を任意の検索サイトで検索できる Chrome 拡張機能。

---

## 特徴

- 任意の検索サイト（Amazon・楽天など）を登録可能
- 右クリックメニューから選択文字列を検索
- サイトリストは自由に追加・削除・並び替え可能
- Chrome のストレージに保存されるため設定は自動で保持

---

## インストール方法（開発者モード）

1. 本リポジトリをダウンロード or `git clone`  
2. Chrome の拡張機能管理画面を開く  
3. 右上の **デベロッパーモード** を ON  
4. **パッケージ化されていない拡張機能を読み込む** をクリック  
5. このフォルダを選択して読み込み

---

## 使い方

1. Web ページで文字列を選択
2. 右クリックメニューに登録した検索サイト名が表示される
3. クリックすると新しいタブで検索結果が表示される

---

## 設定方法

- 拡張アイコンをクリックするとオプション画面が開きます
- 「ADD」で新しい検索サイトを追加
- サイト名をクリックして編集可能
- URL入力欄に検索クエリの URL プレフィックスを設定（例：`https://www.amazon.co.jp/s?k=`）
- 並び替えは上下矢印で可能
- 「CLEAR」で全削除

---

## ファイル構成

<pre>
├─ manifest.json  # 拡張の設定ファイル<br>
├─ options.html   # オプションページ<br>
├─ script.js      # オプションページ用JS<br>
├─ service_worker.js  # コンテキストメニュー管理<br>
├─ style.css      # オプションページ用スタイル<br>
└─ icons/         # アイコン画像<br>
</pre>

---

## 権限

- `contextMenus` : 右クリックメニューの追加  
- `storage` : サイトリスト保存  
- `tabs` : 新しいタブで検索結果を開くため
